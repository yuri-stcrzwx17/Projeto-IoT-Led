#  Projeto Wokwi - Projeto da ESP32 com Led

##  Sobre
Projeto desenvolvido no Wokwi utilizando ESP32.  
O circuito realiza: Ele faz o led ascender e apagar por 2 segundos.

##  Tecnologias e Ferramentas utilizadas
- Wokwi
- C++
- ESP32
- Led 
- Git & GitHub

##  Como executar
1. Abra no Wokwi  
2. Clique em **Start Simulation**  

Ou compile o codigo na IDE correspondente.

##  Arquivos
- sketch.ino / main.py  
- diagram.json  
- wokwi.toml
